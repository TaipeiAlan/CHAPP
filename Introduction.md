# CHAPP — 國語測驗系統功能介紹

> 最後更新：2026-03-20

## 系統概述

CHAPP 是一套純前端、單頁 HTML 的**國中國語字音測驗系統**，不依賴任何外部框架或伺服器。所有題庫資料、樣式、邏輯均內嵌於 `CQuiz.html` 一支檔案中，以瀏覽器直接開啟即可使用。

---

## 目前支援題型

| 題型 | 狀態 | 說明 |
|------|------|------|
| 字音 | ✅ 已實作 | 從句子中辨識目標字的正確注音（BoPoMoFo） |
| 字形 | 🔜 預留 | 辨識正確字形（待開發） |
| 字義 | 🔜 預留 | 辨識正確字義（待開發） |
| 成語 | 🔜 預留 | 成語意義辨析（待開發） |

## 字音測驗流程

```
首頁 → 開始測驗 → 20 題作答 → 批改 → 結果頁 → 歷史記錄
```

### 1. 首頁

- 顯示題庫規模（字音題目總數）
- 若已有作答記錄，顯示：**測驗次數 / 平均分 / 最高分 / 上次測驗日期**
- 兩個入口：「開始測驗」、「歷史記錄」

### 2. 出題

- 從題庫中隨機抽取 **20 題**字音題目
- 每題格式：

  ```
  第 N 題
  句子「目標字」句子 — 下列何者為「　」內字的正確讀音？
  (A) ㄍㄨㄥ   (B) ㄍㄨㄥˋ   (C) ㄍㄨㄥˊ   (D) ㄎㄨㄥ
  ```

- 選完一題後，畫面自動捲動到下一題

### 3. 錯誤選項生成策略（保證三個干擾選項為真實注音）

| 優先順序 | 策略 |
|----------|------|
| 1 | 使用題庫 `wrong_pinyin` 欄位（最常見錯誤） |
| 2 | 相同聲韻、不同聲調（如 ㄍㄨㄥ → ㄍㄨㄥˋ） |
| 3 | 相同韻母、不同聲母（換聲母） |
| 4 | 相同聲母群內替換（ㄍ↔ㄎ↔ㄏ） |
| 5 | 隨機從題庫其他注音備用 |

### 4. 批改

- 所有題目填完才能批改（點擊「批改測驗」後直接送出）
- **若有漏填**：點擊「批改測驗」會以紅框動畫標示第一個未填題，並自動捲動至該題
- 批改結果即時存入 `localStorage`

### 5. 結果頁

- **摘要卡片**：
  - 大圓圈顯示得分（如 18 / 20）
  - 評語（依得分比例）
  - **20 格答題概覽**：每格顯示題號及 ✓ / ✗，可點擊跳至該題詳細說明
- **詳細題目列表**：每題顯示選項高亮（綠色=正確，紅色=錯誤），答錯時顯示正確答案與字義解析

### 6. 歷史記錄

- 儲存於 `localStorage`，key：`Cquiz_history`，最多保留 50 筆
- 歷史列表顯示：日期、分數、百分比長條圖
- 可點擊任意記錄重新查看詳細批改結果
- 可一鍵清除所有記錄

## 題庫來源

`CQBank.csv`（pipe 分隔），共 1575 筆，欄位說明：

| 欄位 | 說明 |
|------|------|
| `sentence` | 例句或詞語 |
| `target_char` | 被測字 |
| `correct_pinyin` | 正確注音 |
| `wrong_pinyin` | 常見錯誤注音（hint） |
| `type` | 題型標記（音 / 義 / 形 / 成，可多重） |
| `meaning` | 正確字義（批改解析用） |
